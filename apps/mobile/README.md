# Wave Forecast - Mobile App

React Native (Expo) mobile application for wave forecasting.

## Getting Started

```bash
pnpm install
pnpm start
```

## Tech Stack

- Expo (React Native)
- Expo Router
- TypeScript
- Shared API client from `@wave-forecast/api-client`

## Development

- Press `i` for iOS simulator
- Press `a` for Android emulator
- Scan QR code with Expo Go app for physical device

## Deployment

Built and deployed via EAS (Expo Application Services):

```bash
pnpm build:ios
pnpm build:android
```
