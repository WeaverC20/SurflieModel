# Wave Forecast - Web App

Next.js web application for wave forecasting.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS (to be added)
- Shared API client from `@wave-forecast/api-client`

## Deployment

Deployed to Vercel via automatic git integration.
